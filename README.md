# AgencyKart 🚀  
### Scale Smarter

AgencyKart is an interactive, scroll-based startup pitch presentation built using Streamlit.

This project presents our disruptive entrepreneurship idea — an AI-powered B2B marketplace that intelligently matches businesses with the right agencies.

---

## 🌟 Project Overview

AgencyKart solves the inefficiency in how startups and SMEs connect with agencies in India.

Instead of relying on guesswork and referrals, AgencyKart uses:

- AI-powered brief submission
- Intelligent agency matching
- Structured proposal system
- Secure milestone-based payments

Our mission: **Scale Smarter.**

---

## 🖥️ Tech Stack

- Python
- Streamlit
- Pillow (for logo image rendering)
- Custom CSS animations
